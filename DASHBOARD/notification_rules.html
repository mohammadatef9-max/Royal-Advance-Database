🔔 Notifications

Create and manage Telegram notification rules. A rule watches a record type, fires when a condition is met, and sends a templated message to the group chat or to a permission group — no code needed.
How it works: pick a source (a record type we watch), choose when it should fire, write the message using the {{field}} chips, and pick who receives it. Use Test to send a sample to yourself/group before enabling. New source types (brand-new tables) need a one-time wiring by an admin/developer.
